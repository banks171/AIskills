# Claude Code Skills Collection

这是一个 Claude Code 技能集合，包含多种专业领域的技能扩展，用于增强 Claude Code 的能力。

| 技能 | 描述 |
|------|------|
| [clickhouse-architect](./clickhouse-architect/) | ClickHouse 架构设计与优化 |
| [clickhouse-cloud-management](./clickhouse-cloud-management/) | ClickHouse 云管理 |
| [sql-expert](./sql-expert/) | SQL 数据库专家 |
| [redis-patterns](./redis-patterns/) | Redis 缓存与数据结构模式 |
| [influxdb-timeseries](./influxdb-timeseries/) | InfluxDB 时序数据库 |

## 使用方法

将此仓库克隆到 `~/.claude/skills/` 目录即可使用：

```bash
git clone https://github.com/banks171/AIskills.git ~/.claude/skills
```
